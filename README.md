# Data Quality Analysis of Brazilian E-commerce Orders Dataset

## Overview
This project evaluates the data quality of a Brazilian e-commerce orders dataset using Python and Pandas.

The analysis focuses on:

- dataset completeness
- uniqueness of identifiers
- missing data patterns
- imputation strategies
- visualization of purchase behaviour

## Dataset
Brazilian e-commerce orders dataset containing ~99,000 transactions.

## Tools Used
Python  
Pandas  
Matplotlib  
Google Colab

## Analysis Steps
1. Data profiling
2. Missing value analysis
3. Completeness and uniqueness metrics
4. Visualization of purchase time patterns
5. Evaluation of imputation strategies

## Key Findings
Missing values were mainly associated with cancelled or incomplete orders rather than data errors.

Replacing missing values with "Unknown" preserved the dataset while maintaining transparency.

## Visualizations
Purchase hour histogram and boxplot were created to explore customer activity patterns.
